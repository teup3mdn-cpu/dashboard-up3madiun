Dashboard kinerja UP3 Madiun

Ringkasan kinerja operasional untuk pengambilan keputusan — susut, penjualan, P2TL, dan prognosa

Format (5 kolom, urut persis seperti kolom C–G di sheet sumber): Bulan, Target_Bulanan, Realisasi_Bulanan, Target_Kumulatif, Realisasi_Kumulatif. Kolom diambil berdasarkan urutan posisi setelah kolom Bulan (bukan nama header), supaya tidak salah tangkap seperti sebelumnya. Realisasi_Bulanan adalah nilai s.d hari ini untuk bulan berjalan (link langsung ke sumber, ter-update real-time).
